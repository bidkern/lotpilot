# Portfolio Overview: LotPilot

LotPilot is a dealer operations platform for inventory onboarding, source sync, listing review, exports, and customer messaging. It is the strongest portfolio proof for operations, implementation, product operations, customer operations, sales operations, and workflow automation roles.

## What this project proves

- Can map a real business workflow from intake to review, execution, and follow-up.
- Can think through messy operational edge cases instead of assuming perfect data.
- Can build a multi-step SaaS-style interface with inventory, employees, approvals, exports, and messaging.
- Can structure a larger TypeScript/Next.js project with Prisma, background jobs, validation, and audit-friendly records.
- Can document a product in a way a non-engineering reviewer can understand quickly.

## Reviewer path

1. Start with the screenshots in [README.md](README.md).
2. Review the onboarding flow and inventory dashboard entry points listed in the README.
3. Scan `prisma/schema.prisma` to see the operational data model.
4. Read the verification section to see the project has been linted and built.

## Role fit

LotPilot is best aligned with jobs where the work includes translating business workflows into tools, improving operational follow-through, coordinating messy records, or supporting technical implementation.

The project shows practical product thinking, structured operations, and enough technical execution to work well with engineering, data, and business teams.
